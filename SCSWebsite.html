menu_book Computer Science Curriculum

code Programming Fundamentals

Learn the basics of programming including variables, data types, control structures, and functions. This foundational course covers essential programming concepts using modern languages like Python and Java.

storage Data Structures

Explore fundamental data structures such as arrays, linked lists, stacks, queues, trees, and graphs. Understand how to choose and implement the right data structure for efficient problem-solving and algorithm design.

trending_up Algorithms

Master algorithm design and analysis including sorting, searching, dynamic programming, and graph algorithms. Learn Big O notation and complexity analysis to write efficient solutions to real-world problems.

category Object-Oriented Programming

Understand the principles of OOP including classes, objects, inheritance, polymorphism, and encapsulation. Apply these concepts to design scalable and maintainable software systems using languages like Java and C++.

language Web Development

Build dynamic and responsive web applications using HTML, CSS, JavaScript, and modern frameworks. Learn front-end and back-end development along with web design principles and user experience best practices.

database Databases

Design and manage databases using SQL and NoSQL systems. Understand data modeling, normalization, indexing, and query optimization. Learn about database security and managing large-scale data systems.

engineering Software Engineering

Apply software development methodologies including SDLC, Agile, and DevOps. Learn about requirements gathering, design patterns, testing strategies, and project management for building professional software.

cloud_queue Computer Networks

Study network fundamentals including TCP/IP protocols, OSI model, routing, and network security. Understand how computers communicate over networks and design efficient network architectures for data transmission.

security Cybersecurity

Protect systems from cyber threats by learning about encryption, network security, ethical hacking, and vulnerability assessment. Understand security principles and implement secure coding practices.

smart_toy Artificial Intelligence & Machine Learning

Explore AI and ML concepts including supervised learning, neural networks, natural language processing, and computer vision. Apply machine learning algorithms to solve real-world problems and build intelligent systems.
